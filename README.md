#  PIXELATE — Plataforma de Divulgação de Eventos 

A Pixelate é uma plataforma voltada para a divulgação, organização e participação em eventos do meio gamer. Nosso objetivo é centralizar informações, facilitar o acesso a eventos e aproximar comunidade e organizadores por meio de uma interface simples e acessível.

---

##  Introdução

Muitos eventos culturais, especialmente os ligados à comunidade gamer, não recebem a visibilidade necessária para alcançar seu público. Isso gera baixa participação, reduz o engajamento e dificulta o crescimento de comunidades.

A Pixelate surge como solução para esse problema. A plataforma permite:

- Visualizar eventos disponíveis  
- Se inscrever em eventos  
- Criar e divulgar novos eventos
- 
A proposta contribui para ampliar a visibilidade da cultura gamer, facilitar conexões e fortalecer a participação social por meio de eventos acessíveis e organizados.

##  Funcionalidades

###  Perfil do Usuário
- Gerenciar dados pessoais  
- Ver eventos inscritos  
- Ver eventos criados  

###  Cadastro de Eventos
- Criar eventos com data, horário, local e descrição  
- Facilitar divulgação e organização de atividades  

###  Inscrição em Eventos
- Inscrição rápida e prática
